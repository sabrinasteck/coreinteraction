Getting your files organized

index.html is the standard naming convention for the primary HTML file in a directory. The browser will assume that a URL requesting a directory is actually looking for the index.html file. In other words www.website.com/index.html and www.website.com are exactly the same thing to a browser.

Move into a folder by using the following convention "folder name/". For instance, the image below is 2 folders away (relative to this file). So the path is "assets/img/YOURFILESNAME.jpg". See this working below.
